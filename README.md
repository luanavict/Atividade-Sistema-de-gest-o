# Atividade-Sistema-de-gestao
### Um menu para você escolher:
* Sistema de cadastro de aluno (usando listas):
  ##### Esse sistema permite:
  - Cadastrar alunos
  - Listar alunos
  - Buscar um aluno pelo nome
  - Remover alunos
  - Verificação de texto com in
  - Comparação de nomes
  - Transformação de nomes em maiúsculas
  - Remoção do último item
  - Encerramento do sistema
  
* Controle de Despesas Pessoais do aluno:
  ##### Sistema que guarda as despesas do mês.
  - Registrar valores de despesas até que o usuario digite 0.
  - Calcular o total gasto
  - Calcular a média de gastos
  - Mostrar o maior e o menor valor cadastrado
    
* Cardápio da Lanchonete:
  - Mostrar um cardápio com itens e preços (listas paralelas)
  - O cliente escolhe itens digitando o número correspondente
  - Somar o total do pedido
  - Mostrar o resumo final(itens + total a pagar)
* Controle de Estoque da Lanchonete
  ##### Programa que controla produtos de uma loja
  - Adcionar novos produtos (nome e quantidade)
  - Atualizar a quantidade de um produto
  - Listar todos os produtos cadastrados
  - Mostrar produtos com estoque zerado
